Design Tokens for Generative Interfaces

Generative interfaces need token systems that account for dynamic states such as confidence, citations, moderation flags, and collaborative edits.

Token Architecture

  • Response Tokens: typography, spacing, and hierarchy for long-form output.
  • State Tokens: trusted, uncertain, flagged, and needs review states.
  • Interaction Tokens: prompt input, refine action, and citation hover behavior.
  • Safety Tokens: warning and escalation color systems with accessibility contrast.

Implementation Guidance

  • Connect component states directly to eval outcomes where possible.
  • Test token behavior across long answers, short answers, and streamed output.
  • Standardize citation and disclaimer styles across all assistant surfaces.
