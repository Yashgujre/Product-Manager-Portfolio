# Activation Journey Blueprint

## Goal
Move new teams from sign-up to sustained collaboration behavior.

## Journey Steps
1. Workspace creation (self-serve, under 60 seconds).
2. Invite teammates and create first channels.
3. Connect core integrations (Jira/GitHub/CRM equivalents).
4. Reach 2,000 message activation threshold.
5. Trigger expansion prompts for adjacent teams.

## Friction Points to Monitor
- Team invite drop-off.
- Channel setup confusion.
- Integration setup abandonment.
- Notification overload leading to mute/churn.

## Success Signals
- 14-day activation target met.
- DAU/MAU above engagement threshold.
- Retention cohort remains above target.
